## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) 15.2
- **Styling**: Tailwind CSS 3.3 + CSS Modules
- **Animations**: Framer Motion 12.6
- **Content**: MDX with automatic type safety
- **Deployment**: Vercel
